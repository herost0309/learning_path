**对我们团队的意义**：
- 20-30% 的效率提升是**可以期待**的，不是吹牛
- 代码采纳率从 0 到 30% 通常在 2-4 周内实现
- AI 生成代码的质量需要额外关注（churn rate 更高）

## 三、数据收集指南

### 自动化收集（推荐）

- **AI 代码采纳率**: IDE 插件内置统计功能
- **上线周期**: CI/CD 系统 API 自动提取
- **Bug率**: Bug 追踪系统报表
- **测试覆盖率**: 测试工具自动生成

### 手动收集

- **开发者满意度**: 每月发放匿名问卷
- **定性反馈**: 周分享会记录

### 数据质量要求

- 数据收集要一致，同一口径比较
- 不要为了指标好看而做数据
- 关注趋势而非绝对值
- 定性反馈和量化数据结合看
